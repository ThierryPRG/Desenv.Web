Diferença entre Linguagens de Programação Server Side e Client Side

As linguagens de programação podem ser classificadas em dois tipos principais: Server Side e Client Side. Linguagens Server Side, essas linguagens sao feitas em um servidor q abriga um site ou app ela não fica vísivel para os usuário, alguns exemplos sao o PHP e o Python, ja o Client Side as linguagen são processadas no navegador do cliente ou usuário e o código é visível um exemplo é o HTML e JavaScript.
